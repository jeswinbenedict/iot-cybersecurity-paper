# ⚡ IoT Cybersecurity — Hybrid ML/DL Models

> 75 billion IoT devices. Most are unsecured. That's a civilizational risk.

## The Problem
Hackers don't wait. IoT devices are weak, diverse, and everywhere.
Traditional security is dead on arrival in this environment.

## The Solution
Hybrid Machine Learning + Deep Learning that actually works.
Not just on benchmarks. In the real world.

## What's Inside
- **CNN** — detects spatial attack patterns instantly
- **LSTM** — catches slow, multi-stage attack campaigns over time
- **Autoencoders** — flags anomalies without needing labeled attack data
- **Federated Learning** — privacy-first, trains across thousands of devices
- **Edge Deployment** — real-time detection, no cloud dependency

## Key Results
- Detection accuracy **99%+** on UNSW-NB15 and NSL-KDD
- F1 Score **0.97+** across DDoS, brute force, and web attacks
- Zero-day threat detection — no prior signature needed
